Willkommen auf dem Verinice Server

Wenn Sie diese Seiten sehen können, wurde die Basis-Komponente des verinice-Servers (verinice.PRO) erfolgreich installiert!

Je nachdem, welche Komponenten Ihr Administrator installiert hat, sind weitere Module verfügbar, die Sie oben im Menü wiederfinden.

Damit Sie damit beginnen können, verinice auf Ihrem Desktop zu benutzen, müssen Sie einen der Clients installieren, die Sie im Menü unter dem Reiter "Client Downloads" finden.

Bei Fragen und Problemen finden Sie hilfreiche Links und die Kontaktdaten des verinice-Teams unter dem Reiter "Kontakt".

Wir wünschen Ihnen viel Erfolg und Spaß mit verinice.
