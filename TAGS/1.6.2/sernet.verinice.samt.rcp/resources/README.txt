# 2010-05-27 akoderman

This folder should contain the catalogs to be included in the verinice distribution. The must use the usual
verinice CSV format for control catalogs.

You will need at least two files:

- samt-catalog.csv
- samt-catalog_de.csv

Their name must match the filename defined in sernet.verinice.samt.rcp.messages.properties for each locale.
